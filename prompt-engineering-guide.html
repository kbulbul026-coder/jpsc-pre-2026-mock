Core Concepts

🤖

LLM क्या है?

Large Language Model — Transformer architecture पर बना, जो text generate करता है।

🪙

Token

AI word नहीं, tokens पढ़ता है। "Playing" = "play" + "ing" — दो tokens।

🪟

Context Window

AI एक बार में जितना याद रख सकता है — उससे ज्यादा दो तो पुरानी बात भूल जाएगा।

🧠

Attention

Transformer पूरे sentence को एक साथ process करता है — RNN की तरह शब्द-दर-शब्द नहीं।

01

AI का इतिहास — RNN से Transformer तक

RNN

Recurrent Neural Network

समस्या: Vanishing Gradient। लंबा paragraph देते थे — शुरुआत भूल जाता था।

LSTM

Long Short-Term Memory

RNN का upgrade — Gates लगाए कि क्या याद रखना है। लेकिन training बहुत slow।

2017

Transformer — "Attention is All You Need"

Google का paper। पूरा sentence parallel process करता है। यही ChatGPT, Gemini का base है।

Prompt Engineering इसलिए possible है क्योंकि Transformer है। RNN/LSTM में prompt का कोई मतलब नहीं था।

03

Direct vs Structured Prompting

Direct Prompting
Structured Prompting
"Write a blog post about coffee."
AI को नहीं पता — कितने words? किस audience के लिए? क्या tone?
Role + Context + Task + Constraints + Output format
AI को सब पता है — आपका result बेहतर होगा।
RCTO Framework — Structured Prompt Template Role: Act as a Senior Content Marketer with 10 years experience
Context: I'm launching a new organic coffee brand for Gen-Z
Task: Write an engaging blog post title (max 300 words)
Constraints: Avoid jargon. Don't mention caffeine side effects
Output: Markdown format with a catchy H1 header
04

Core Prompting Techniques

①
Zero-Shot

कोई example नहीं — सीधा instruction

"Classify the sentiment: The battery is amazing but screen is dull." → AI खुद decide करता है।

②
Few-Shot

2-3 examples देकर pattern सिखाना

"I love pizza → positive (1), Delivery was late → negative (0)" — फिर नया example दो। Output specific format में मिलेगा।

③
Chain-of-Thought

"Think step by step" — hallucination कम होता है

"A farmer has 15 sheep, all but 8 die. How many left?" → Answer: 8 (not 7)। CoT logical problems में जरूरी है।

④
ReAct Framework

Thought → Action → Observation loop

Research tasks के लिए। AI सोचता है, search करता है, observe करता है — loop में।

⑤
Tree-of-Thoughts

3 experts एक साथ brainstorm करें

"Imagine 3 different experts solving this problem..." — AI खुद best solution चुनता है, weak ideas eliminate करता है।

⑥
Iterative Prompting

पहली बार perfect नहीं — refine करते रहो

Image generation, Resume writing — जब तक desired output न मिले, prompt को improve करते रहो।
