8. Prompt Engineering क्या है?

Prompt Engineering = AI को सही instructions देने की कला।

“Garbage in = Garbage out.”

जितना clear prompt होगा, उतना powerful output मिलेगा।

Weak Prompt

Write an Instagram caption.
  • No audience
  • No tone
  • No context

Result: Generic output

Better Prompt

Act as a luxury travel content creator.

Write a premium Instagram caption for a Maldives resort reel.

Tone:
- Elegant
- Aspirational
- Short

Target Audience:
Indian luxury travelers aged 25-40

Include:
- Hook
- CTA
- 5 hashtags

अब AI audience, tone और structure समझकर बेहतर response देगा।

9. Prompting Techniques

9.1 Role-Based Prompting

Act as a doctor.
Act as a coding mentor.
Act as a marketing strategist.

AI उस role की language और thinking adopt करता है।

9.2 Context Engineering

Bad Prompt

Write a blog.

Better Prompt

Write a beginner-friendly SEO blog
for Indian freelancers
about ChatGPT productivity tools.

9.3 Few-Shot Prompting

Example 1:
Coffee Caption:
Fuel your hustle ☕

Example 2:
Sneaker Caption:
Walk like you own the street 👟

Now create one for wireless headphones.

AI examples देखकर pattern समझता है।

9.4 Chain-of-Thought Prompting

Solve this problem step by step.

1. Identify issue
2. Analyze causes
3. Suggest solutions
4. Prioritize best option

इससे AI reasoning improve करता है।

9.5 Structured Output Prompting

Explain SEO in this format:
- Definition
- Benefits
- Tools
- Examples
- Mistakes

Result:

  • Clean output
  • Less editing
  • Professional structure
